# timecard_C
最大31回（31日分）まで出退勤できるタイムカード。31回を超えて出退勤を登録すると古いデータから消えて更新される。
＜使い方＞
（1）メニュー画面にてID登録の数字（3）を入力し、ID3桁を5人分まで登録。
（2）メニュー画面にて出勤の数字（1）を入力し、一人分のIDを入力。すると出勤登録が完了する。
（3）メニュー画面にて退勤の数字（2）を入力し、一人分のIDを入力。すると退勤登録が完了する。
（4）メニュー画面にて履歴確認の数字（4）を入力し、一人分のIDを入力。するとそのIDの過去の履歴が表示される。
（5）メニュー画面にて終了の数字（5）を入力すると、プログラムが終了する。

※履歴の表示には出退勤の日付、時刻がひょうじされ、勤務時間（ここでは秒）が表示される。
